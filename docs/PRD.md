# Perso.ai QA 자동화 PRD (Product Requirements Document)

## 프로젝트 개요

**목적**: Claude Computer Use를 활용하여 Perso.ai 웹사이트의 20개 QA 태스크를 자동화

**기술 스택**:
- Python 3.12
- Claude Computer Use API (Sonnet 4.5)
- Docker (X11 display)

---

## 태스크 목록

### 대상 서비스
- **서비스명**: PERSO AI Dubbing PC
- **URL**: https://perso.ai/ko/workspace/vt

### 20개 QA 태스크

#### 1. 홈페이지 접속 및 로그인
- **설명**: 로그인 후 `https://perso.ai/ko/workspace/vt`로 이동하는지 확인
- **빈도**: 1일 1회(오전)
- **담당**: 오지연

#### 2. 로컬 파일 업로드
- **설명**: Drag & Drop document files > 로컬 영상 또는 오디오 파일 업로드
- **빈도**: 1일 1회(오전)

#### 3. 유튜브 롱폼 링크 업로드
- **설명**: 유튜브 롱폼 링크를 업로드 했을 때 영상이 정상적으로 업로드 되는지 확인
- **빈도**: 1일 1회(오전)

#### 4. 유튜브 쇼츠 링크 업로드
- **설명**: 유튜브 숏폼 링크를 업로드 했을 때 영상이 정상적으로 업로드 되는지 확인
- **빈도**: 1일 1회(오전)

#### 5. 틱톡 링크 업로드
- **설명**: 틱톡 링크를 업로드 했을 때 영상이 정상적으로 업로드 되는지 확인
- **빈도**: 1일 1회(오전)

#### 6. 틱톡 라이트 링크 업로드
- **설명**: 틱톡 라이트 링크를 업로드 했을 때 영상이 정상적으로 업로드 되는지 확인
- **빈도**: 1일 1회(오전)

#### 7. 구글 드라이브 링크 업로드
- **설명**: 구글 드라이브 링크를 업로드 했을 때 영상이 정상적으로 업로드 되는지 확인
- **빈도**: 1일 1회(오전)

#### 8. 영상 생성 시 RU 차감
- **설명**: 영상 업로드 후 원본 언어와 번역 언어를 선택하고 편집을 진행했을 때 RU가 정상적으로 차감되는지 더빙 admin 확인
- **빈도**: 1일 1회(오전)

#### 9. 업로드 후 최근 비디오에 프로젝트 생성 결과물 확인
- **설명**: [립싱크 옵션 선택] 입모양이 알맞게 적용이 되었는지 확인(미선택 and 선택)
- **빈도**: 1일 1회(오전)

#### 10. 생성된 결과물에 알맞게 뱃지가 붙었는지 확인
- **설명**: 생성된 결과물이 립싱크 인경우 립싱크 뱃지가 붙었는지 확인 / 언어가 알맞게 붙었는지 확인
- **빈도**: 1일 1회(오전)

#### 11. 자막 다운로드
- **설명**: 자막 파일 다운로드가 되는지 확인 (원본,번역 언어로 다운로드 했는지 파일 내용 확인 필요)
- **빈도**: 1일 1회(오전)

#### 12. 오디오 다운로드
- **설명**: 오디오 파일 다운로드가 되는지 확인 (3가지 중 1개 랜덤 시도)
- **빈도**: 1일 1회(오전)

#### 13. 영상 다운로드
- **설명**: 영상 파일 다운로드가 되는지 확인
- **빈도**: 1일 1회(오전)

#### 14. 자막 스크립트 확인
- **설명**: 원본 영상의 언어와 번역 영상의 언어가 원본 언어,타겟 언어에 알맞게 STT 되었는지 확인
- **빈도**: 1일 1회(오전)

#### 15. 프로젝트 편집 확인
- **설명**: 원본 언어 내용 수정 후 재번역 버튼을 눌렀을 때 타겟 언어 내용 바꾸는지 확인
- **빈도**: 1일 1회(오전)

#### 16. 프로젝트 편집 확인
- **설명**: 타겟 언어 내용 수정 후 포커스 아웃했을 때 수정사항 저장되는지 확인
- **빈도**: 1일 1회(오전)

#### 17. Community Spotlight 영상 재생
- **설명**: Community Spotlight의 영상들이 정상적으로 재생이 되는지 확인
- **빈도**: 1일 1회(오전)

#### 18. 캐러셀 공지 영역 링크 이동 확인
- **설명**: 개별 공지가 커뮤니티 허브URL로 정상적으로 연결되는지 확인
- **빈도**: 1일 1회(오전)

#### 19. 우측 하단 허브스팟 배너
- **설명**: 우측 하단 허브 스팟 배너가 정상적으로 뜨는지 확인
- **빈도**: 1일 1회(오전)

#### 20. 허브스팟 배너 언어 확인
- **설명**: 우측 하단 허브 스팟 배너가 언어에 맞게 뜨는지 확인 (한국어/영어)
- **빈도**: 1일 1회(오전)
